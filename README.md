# 🏷️ Online Auction System (C++)

A simple **console-based Online Auction System** developed in **C++** that simulates a real-world bidding process.  
Users can place bids on an item, track the highest bid, and the system declares the winner when the auction ends.

---

## 📌 Features

- Enter auction **item name and base price**
- Place **multiple bids**
- Automatically **track highest bid**
- Validate bids using **comparison logic**
- Display **current highest bidder**
- **Declare winner** when auction ends
- Menu-driven **interactive console program**

---

## ⚙️ Technologies Used

- **Programming Language:** C++
- **Concepts Applied:**
  - Variables & Data Types
  - Conditional Statements (`if-else`)
  - Switch Case
  - Loops (`do-while`)
  - String Handling
  - Input / Output Operations

---

## 🧠 Program Logic

1. User enters the **item name** and **base price**.
2. System sets the **base price as the initial highest bid**.
3. Menu options are displayed:
   - Place Bid
   - View Highest Bid
   - Exit
4. When a user places a bid:
   - Bidder name and amount are entered.
   - Program compares it with the current highest bid.
5. If the bid is **higher**, the system updates the highest bidder.
6. When the user selects **Exit**, the system declares the **auction winner**.
